# Agentic RAG for Agile Task Management

This repository presents the core implementation of an Agentic Retrieval-Augmented Generation (RAG) system for Agile project task management analysis.

## Features

- Semantic retrieval from Agile project data
- Agent-based reasoning
- Query rewriting
- Relevance grading
- Conversational memory

## Technologies

- LangChain
- LangGraph
- OpenAI GPT
- Vector Database
